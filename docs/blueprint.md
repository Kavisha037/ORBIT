# **App Name**: SkillFlow

## Core Features:

- Onboarding: Collect user data: name, timetable (upload/input), and preferred skill path.
- Schedule Integration: Parse the user's timetable to identify available time slots.
- Personalized Daily Plan: An AI-powered tool automatically creates a personalized daily plan by slotting skill-building tasks into the user's free time.
- Progress Tracking: Monitor task completion daily and track progress via streaks, levels, and achievements.
- Daily Check-in: Prompt the user with check-in questions regarding task completion and collect input.
- Adaptive Learning: Modify task difficulty, duration, and future plans based on daily check-in responses. For example, schedule the same missed task, or adapt and replace that task.
- Dashboard Display: Present the daily schedule clearly, showing both academic commitments and AI-generated skill-building tasks.

## Style Guidelines:

- Primary color: Light, calming blue (#A0D2EB) to evoke focus and serenity.
- Background color: Very light blue (#F0F8FF) to promote clarity.
- Accent color: Muted orange (#E59866) to highlight calls to action without overwhelming.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and readable user experience.
- Use simple, outline-style icons for skills and tasks to maintain a minimalistic aesthetic.
- Maintain a clean and spacious layout with generous use of whitespace to avoid a cluttered feel.
- Incorporate subtle transitions and animations on task completion and progress updates to provide positive feedback.
